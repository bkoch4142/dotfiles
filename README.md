# Setup 

## Installing stow
Using stow as a symlink manager
`sudo apt-get install stow`

## Syncing
```
cd ~
git clone https://github.com/bkoch4142/dotfiles
cd dotfiles
stow vim  #make sure files you stow don't already exist
```
